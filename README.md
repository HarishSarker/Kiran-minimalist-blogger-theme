## How to Install
1. Download the `minimalist-blogger-theme.xml` file.
2. Go to your [Blogger Dashboard](https://www.blogger.com).
3. Navigate to **Theme** > Click the arrow next to **Customize**.
4. Select **Restore** > **Upload**.
5. Choose the XML file you downloaded.
